# Weeks 1–4 — Detailed Daily Schedule
### MERN Developer + AI Learner + Job Seeker

> You have 3 goals running in parallel:
> 1. Get a MERN job (income now)
> 2. Build AI foundation (income later)
> 3. Start monetizing skills (income bridge)
>
> This schedule makes all 3 work together without burning you out.

---

## Realistic Daily Time Budget

| Block | Hours |
|---|---|
| Sleep | 7 hrs |
| Morning routine, meals, breaks | 2 hrs |
| **MERN Job Search + Interview Prep** | **3 hrs** |
| **AI Learning + Building** | **5 hrs** |
| **Content + Monetization** | **1.5 hrs** |
| **Planning + Review** | **0.5 hrs** |
| **Total active work** | **10 hrs/day** |

> 10 hours/day is hard but sustainable. 12 will burn you out while also job hunting.
